1. difference between flex row and column ?

This is the css property. it based on the concept of flex direction. flex-row is a default value and it is used to display child containers horizontally based on parent container. flex-column is used to display child container vertically based on parent container. Example: flex-direction: row; flex-direction: column;

2. Explain Inline, Internal and External Style Sheets ?

Inline css is a type of css style sheet. it directly applied to particular or unique element in a HTML,by using style attribute. Internal css is a type of css style sheet. will define by using style tag inside head tag. this unique style to same typr of html elements or class or id in single html document. External style is a type of css style sheet. external have separate css file to add a style to html element. can use this external style as much as want in any number of html document. to define external css we use link tag inside head tag.

4. Difference between absolute and relative positioning ?

relative position palces the element in the normal position. without changing the layout around it. absolute position palces the element relative to its parent position. and changing the layout around element.

5. What is grid-template columns used for ?

grid-template-columns is a property, it takes maxmimum width of column according to parent container(grid-layout). the values are a space separated list, where each value specifies the size of the respective column.

6. What is the z-index in CSS ?

z-index is a stack level of property in css. and it is stack order of the elements. greater value will be top and lower value will be below. z-index will work with positions only. z-index value can be a negative and positive.

7. Difference between Margin and Padding ?

margin and padding is css properties. where we provide space between and around the elements. both work all 4 directions like top, right, bottom, left. the value of margin and padding will be use pixel. margin provide space outside the border, and near my element. padding provide space inside border and ouside the content.
